# Actividad 5 - Machine Learning con MLflow

## Objetivo

Desarrollar un flujo completo de Machine Learning utilizando el dataset Titanic, aplicando buenas prácticas de preparación de datos, entrenamiento de modelos, optimización mediante GridSearchCV y seguimiento de experimentos con MLflow.

## Modelos implementados

- Regresión Logística
- Random Forest

## Técnicas utilizadas

- Preprocesamiento de datos
- Pipeline de Scikit-Learn
- GridSearchCV
- Validación cruzada
- MLflow
- Versionamiento mediante SHA-256

## Herramientas

- Python
- Google Colab
- Scikit-Learn
- Pandas
- MLflow
